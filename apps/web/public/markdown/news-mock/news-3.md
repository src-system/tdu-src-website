# 旭祭2024出展レポート

11月23日〜24日に開催された東京電機大学の学園祭「旭祭2024」に出展しました。

## 展示内容

### ゲームプログラミング班

今年制作した新作ゲーム3タイトルを展示しました。

- **ダンジョンクエスト** - ローグライクRPG
- **スペースシューター** - 縦スクロールSTG
- **パズルマスター** - 脳トレパズルゲーム

### 3D班

VRヘッドセットを使った3D作品体験コーナーを設置しました。

### 2D班・デザイン班

イラスト展示とグッズ販売を行いました。

## 来場者数

2日間で約500名の方にご来場いただきました。
