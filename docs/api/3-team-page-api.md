# `/team` - Teamページ

班の一覧を表示するページです。

## ページヘッダー（トップ画像）

### 概要
Teamページ上部に表示されるヘッダー画像とタイトルです。

### フィールド
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| image | image | ✓ | ヘッダー画像（16:9推奨） |
| alt | string | ✓ | 画像の代替テキスト |
| title | string | ✓ | ページタイトル |

## 班一覧

### 概要
ソフトウェア研究部の各班を紹介するセクションです。

### 使用箇所
- Teamページ（メインコンテンツ）
- トップページ（Teamセクション）

### フィールド
| フィールド | 型 | 必須 | デフォルト値 | 備考 |
|-----------|-----|------|------------|------|
| title | string | - | "TEAM" | セクションタイトル |
| subtitle | string | - | "班紹介" | サブタイトル |
| description | string | ✓ | - | 説明文 |
| teams | TeamCard[] | ✓ | - | 班カードの配列（6つ必須） |

**TeamCard:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| teamName | TeamType | ✓ | 班名 |
| image | image | ✓ | 班のイメージ画像（16:9推奨） |
| alt | string | ✓ | 画像の代替テキスト |
| description | string | ✓ | 班の簡単な説明 |
| activityLabels | activityLabel[] | ✓ | 班の簡潔な活動例のラベル（1つ以上） |

**activityLabel:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| label | string | ✓ | 活動ラベル（例：「Unity開発」「Web制作」等） |

**TeamType:**
- `game` - ゲームプログラミング
- `web` - Webアプリ
- `sound` - サウンド
- `2d` - 2D
- `3d` - 3D
- `design` - デザイン


**注意:**
- 6つの班すべてが必須
- トップページでも同じデータを使用
- TeamTypeがそのままページ名（URL）になります

# `/team/[teamName]` - 班詳細ページ

各班の詳細情報を表示するページです。

## 1.ページヘッダー（トップ画像）

### 概要
班詳細ページ上部に表示されるヘッダー画像とタイトルです。

### フィールド
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| image | image | ✓ | ヘッダー画像（16:9推奨） |
| alt | string | ✓ | 画像の代替テキスト |

## 2.Aboutセクション

### フィールド
| フィールド | 型 | 必須 | デフォルト値 | 備考 |
|-----------|-----|------|------------|------|
| title | string | - | "ABOUT" | セクションタイトル |
| subtitle | string | ✓ | - | サブタイトル (◯◯班とは？) など |
| description | markdown | ✓ | - | 班の詳細説明（Markdown対応） |
| images | ImageCard[] | ✓ | - | 1~3枚を予定 |

**ImageCard:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| image | image | ✓ | 画像（4:3～16:9推奨） |
| alt | string | ✓ | 代替テキスト |

## 3.班長インタビュー

### 概要
班長のインタビュー記事を表示します。

### フィールド
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| title | string | - | インタビュータイトル（デフォルト: "班長インタビュー"） |
| leaderName | string | ✓ | 班長の名前 |
| leaderImage | image | - | 班長の写真（オプション） |
| content | markdown | ✓ | インタビュー内容（Markdown対応） |


## 4.使用ソフトウェア

### 概要
班で使用している主なソフトウェア・技術スタックを表示します。

### フィールド
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| title | string | - | セクションタイトル（デフォルト: "使用ソフトウェア"） |
| software | SoftwareItem[] | - | ソフトウェアの配列（オプション） |

**SoftwareItem:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| name | string | ✓ | ソフトウェア名 |
| description | string | - | 用途の説明（オプション） |
| icon | image | - | アイコン画像（オプション） |

## 5.ギャラリー

### 概要
班の活動写真や作品画像のギャラリーです。

### フィールド
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| title | string | - | セクションタイトル（デフォルト: "ギャラリー"） |
| images | GalleryImage[] | - | ギャラリー画像の配列（オプション） |
| videos | GalleryVideo[] | - | ギャラリー動画の配列（オプション） |
| sounds | GallerySound[] | - | ギャラリー音源の配列（オプション） |

**GalleryImage:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| image | image | ✓ | 画像 |
| alt | string | ✓ | 画像の代替テキスト |
| authors | string[] | ✓ | 作者（複数可能） |
| caption | string | - | キャプション（オプション） |
| link | string | - | リンクURL（オプション） |

**GalleryVideo:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| video | video | ✓ | 動画 |
| title | string | ✓ | 動画のタイトル |
| authors | string[] | ✓ | 作者（複数可能） |
| thumbnail | image | - | サムネイル画像（オプション） |
| caption | string | - | キャプション（オプション） |
| link | string | - | リンクURL（オプション） |

**GallerySound:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| sound | audio | ✓ | 音源ファイル |
| title | string | ✓ | 曲名・音源タイトル |
| authors | string[] | ✓ | 作者（複数可能） |
| thumbnail | image | - | サムネイル画像（オプション） |
| duration | string | - | 再生時間（例: "3:45"）（オプション） |
| caption | string | - | キャプション（オプション） |
| link | string | - | リンクURL（オプション） |

**注意:**
- すべてのセクション（班長インタビュー、ソフトウェア、ギャラリー）はオプション
- データがない場合は非表示
- ギャラリーは画像・動画・音源を混在して表示可能
