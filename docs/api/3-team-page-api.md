# `/team` - Teamページ

班の一覧を表示するページです。

## 1.ページヘッダー（トップ画像）

### 概要
Teamページ上部に表示されるヘッダー画像とタイトルです。

### フィールド
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| image | image | ✓ | ヘッダー画像（16:9推奨） |
| alt | string | ✓ | 画像の代替テキスト |
| title | string | ✓ | ページタイトル |

## 班一覧

### 2.Aboutセクション
ソフトウェア研究部の班活動についてを説明するセクションです。

### 使用箇所
- Teamページ（Aboutセクション）

### フィールド
| フィールド | 型 | 必須 | デフォルト値 | 備考 |
|-----------|-----|------|------------|------|
| title | string | - | "TEAM" | セクションタイトル |
| subtitle | string | - | "班紹介" | サブタイトル |
| description | string | ✓ | - | 説明文 |

### 3.TeamCardセクション
チームのそれぞれの特徴を説明するセクションです。

### 使用箇所
- Teamページ（TeamCardセクション）

**TeamCard:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| teamName | TeamType | ✓ | 班名 |
| image | image | ✓ | 班のイメージ画像（16:9推奨） |
| alt | string | ✓ | 画像の代替テキスト |
| description | string | ✓ | 班の簡単な説明 |
| activityLabels | activityLabel[] | ✓ | 班の簡潔な活動例のラベル（1つ以上） |

**activityLabel:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| label | string | ✓ | 活動ラベル（例：「Unity開発」「Web制作」等） |

**TeamType:**
- `game` - ゲームプログラミング
- `web` - Webアプリ
- `sound` - サウンド
- `2d` - 2D
- `3d` - 3D
- `design` - デザイン


**注意:**
- 6つの班すべてが必須
- トップページでも同じデータを使用
- TeamTypeがそのままページ名（URL）になります

# `/team/[teamName]` - 班詳細ページ

各班の詳細情報を表示するページです。

## 1.ページヘッダー（トップ画像）

### 概要
班詳細ページ上部に表示されるヘッダー画像とタイトルです。

### フィールド
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| image | image | ✓ | ヘッダー画像（16:9推奨） |
| alt | string | ✓ | 画像の代替テキスト |

## 2.Aboutセクション

### フィールド
| フィールド | 型 | 必須 | デフォルト値 | 備考 |
|-----------|-----|------|------------|------|
| title | string | - | "ABOUT" | セクションタイトル |
| subtitle | string | ✓ | - | サブタイトル (◯◯班とは？) など |
| description | markdown | ✓ | - | 班の詳細説明（Markdown対応） |
| images | ImageCard[] | ✓ | - | 1~3枚を予定 |

**ImageCard:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| image | image | ✓ | 画像（4:3～16:9推奨） |
| alt | string | ✓ | 代替テキスト |

## 3.班長インタビュー

### 概要
班長のインタビュー記事を表示します。

### フィールド
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| title | string | - | インタビュータイトル（デフォルト: "班長インタビュー"） |
| leaderName | string | ✓ | 班長の名前 |
| leaderImage | image | - | 班長の写真（オプション） |
| content | markdown | ✓ | インタビュー内容（Markdown対応） |


## 4.使用ソフトウェア

### 概要
班で使用している主なソフトウェア・技術スタックを表示します。

### フィールド
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| title | string | - | セクションタイトル（デフォルト: "使用ソフトウェア"） |
| software | SoftwareItem[] | - | ソフトウェアの配列（オプション） |

**SoftwareItem:**
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| name | string | ✓ | ソフトウェア名 |
| description | string | - | 用途の説明（オプション） |
| icon | image | - | アイコン画像（オプション） |

## 5.ギャラリー（班ごとの定義）

### 概要
各班ごとに定義される、班の活動写真や作品画像のギャラリーです。各Team（班）が独自のギャラリーコンテンツを持ちます。

### フィールド（各班のTeamsドキュメントに定義）
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| gallery | GalleryItem[] | - | 班ごとのギャラリーアイテム配列（画像・動画・音声を混在可能） |

**GalleryItem:**（typeで種別を判別）
| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| type | "image" \| "video" \| "sound" | ✓ | メディア種別 |
| media | upload | ✓ | メディアファイル（画像/動画/音声） |
| alt | string | ✓ | 代替テキスト |
| thumbnail | image | - | サムネイル（動画・音声の場合） |
| title | string | - | タイトル（音声の場合：曲名等） |

**注意:**
- すべてのセクション（班長インタビュー、ソフトウェア、ギャラリー）はオプション
- データがない場合は非表示
- ギャラリーは各班（Team）ごとに個別に定義され、画像・動画・音源を混在して表示可能
