# ウェブサイトをリニューアルしました

2025年1月6日、ソフトウェア研究部のウェブサイトを全面リニューアルしました。

## リニューアルのポイント

### モダンなデザイン

Next.js 14とTailwind CSSを使用し、モダンで見やすいデザインに生まれ変わりました。

### レスポンシブ対応

スマートフォンやタブレットでも快適にご覧いただけるよう、レスポンシブデザインを採用しています。

### 高速な表示

最新の技術を使用することで、ページの読み込み速度が大幅に向上しました。
