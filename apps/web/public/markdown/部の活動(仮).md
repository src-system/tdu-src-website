# ソフトウェア研究部について

ソフトウェア研究部（通称：ソフ研）は、**プログラミング**や**ゲーム制作**、**Web開発**などを中心に活動しているサークルです。

初心者から経験者まで、幅広いメンバーが所属しており、お互いに教え合いながら技術を磨いています。

## 主な活動内容

### ゲーム制作

UnityやUnreal Engineなどのゲームエンジンを使用して、オリジナルゲームを制作しています。学園祭では毎年制作したゲームを展示・公開しています。

### Webアプリケーション開発

ReactやNext.jsなどのモダンなフレームワークを使って、Webサイトやアプリケーションを開発しています。このサイトもソフ研メンバーが制作しました。

### 競技プログラミング

AtCoderやPaizaなどの競技プログラミングコンテストに参加しています。アルゴリズムやデータ構造を学び、問題解決能力を高めています。

### 勉強会・LT会

定期的に勉強会やLT（ライトニングトーク）会を開催しています。メンバー同士で知識を共有し、新しい技術に触れる機会を設けています。

## 活動日・活動場所

- **活動日**: 毎週 **水曜日** と **土曜日**
- **活動場所**: 千住キャンパス 部室棟

活動日以外でも、Discordでのオンライン交流や自主的な活動が活発に行われています。
